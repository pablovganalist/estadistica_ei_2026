## 🏛️ El Imperativo Internacional (El "Filtro Kissinger")
Este no es un curso de estadística para la política doméstica. La profesión del analista internacional exige observar el mundo allí donde la autoridad soberana termina y comienza la anarquía del sistema. **En este curso, la estadística está subordinada a la comprensión de fenómenos transnacionales, estructurales y geopolíticos.**

Toda evaluación cuantitativa producida aquí debe superar el **Filtro Kissinger**: *¿Proporciona este reporte evidencia empírica para tomar una decisión sobre política exterior, seguridad, o economía política internacional; o sus conclusiones solo son de utilidad para una administración local?* ---

## 💻 Metodología y Uso del Repositorio
A través del modelo de **Aprendizaje Basado en Problemas (ABP)**, este curso divide su carga en dos dimensiones integradas:
1. **Diseño estratégico (Jueves - Aula):** Debate teórico, arquitectura del problema y diseño metodológico.
2. **Ejecución técnica (Viernes - Laboratorio):** Trabajo práctico orientado a la programación y limpieza de datos usando de forma exclusiva **RStudio**.

**¿Cómo usar este GitHub?**
Este repositorio es la biblioteca central del curso. Cada semana tiene su propia carpeta. Haz clic en los enlaces del cronograma para descargar las presentaciones del aula (jueves), los *scripts* de código y las bases de datos (viernes). Las entregas de evaluaciones se realizarán a través de la plataforma institucional de la universidad.

## 🤖 Política de IA Generativa
El uso de asistentes de inteligencia artificial (para limpieza de código o redacción básica) está permitido, siempre que el estudiante sea capaz de auditar, explicar y responsabilizarse —de manera académica y argumentativa— por cada línea de su análisis final.
